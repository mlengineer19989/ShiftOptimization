# APIサーバーの起動について
起動方法は以下のいずれか。
- main.pyを実行。
- コマンド`uvicorn main:app --reload`を実行する。

# 最適化APIの実行
実行方法は以下のいずれか。
- http://127.0.0.1:8000/docs にアクセスして、ブラウザ上で実行する。
- 以下のコマンドを実行する。

```
curl -X 'POST' \
  'http://127.0.0.1:8000/optimize/?n_member_day=[メンバ数の引数]' \
  -F 'upload_file=@[アップロードするファイルのパス]' \
  -o '[出力結果の保存先ファイルパス]'
```
